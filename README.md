OCMOD Helper for Sublime Text 4

Удобный инструмент для разработчиков OpenCart, который позволяет мгновенно создавать структуру OCMOD модификатора из выделенного кода.

Установка

Через Package Control
1. Нажмите Ctrl+Shift+P -> Package Control: Add Repository
2. Вставьте ссылку: https://github.com/kjpower/Ocmod-generator-for-Sublime
3. Нажмите Ctrl+Shift+P -> Package Control: Install Package
4. Найдите ocmod_gen и установите
5. В контекстом меню должно появится OCMOD: Создать модификатор
6. Выделяем текст и выбираем этот пункт меню
7. В буфер обмена помещен сгенерированный окмод
